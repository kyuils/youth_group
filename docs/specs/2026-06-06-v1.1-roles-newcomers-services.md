## 3. 권한별 화면 흐름

### 3.1 teacher (일반 교사)
- v1.0과 동일
- 탭: 출석 / 기도제목 / 기록 / 통계
- 본인 반만

### 3.2 newcomer_staff (새가족부 총괄, 윤희윤)
- 추가 탭: **새가족부**
- 새가족부 탭에서 가능한 작업:
  - 새가족 명단 (P컬럼이 윤희윤인 학생 또는 [새가족] 태그)
  - 학생별 출석 5옵션 + 4주 교육 진도 체크
  - **+ 새가족 추가** 버튼 → 양식 → 시트 append
  - **등반** 버튼 → 반 선택 다이얼로그 → 확정

### 3.3 admin (관리자, 5명)
- 추가 탭: **관리자**
- 관리자 탭에서 가능한 작업:
  - 전체 반 선택 드롭다운 (반 변경하여 다른 반 출석·기도제목 열람)
  - 전체 출석 통계 (반별·주차별 표)
  - 전체 기도제목 (반별 그룹화)
  - 본인이 teacher 권한도 가지면 본인 반 입력 화면도 유지
- 본인이 새가족 처리도 가능 (newcomer_staff 권한 포함)

## 5. 데이터 마이그레이션 (1회)

### 5.1 ATTENDANCE 컬럼 추가 + 상태 변환

기존 데이터 (현재 1행만 있음):
- `날짜 | 학생id | 학생이름 | 반 | 상태 | 기록자email | 기록시각`
- 데이터: `2026-06-06 | 214 | 김단아 | 황규일 | s2 | kyuils@gmail.com | ...`

마이그레이션:
1. F열(기록자email)을 G로, G열(기록시각)을 H로 이동
2. F열 신설 → "기타내용" 헤더
3. E열(상태) 값 변환:
   - `s2` → `P2`, `s3` → `P3`, `s4` → `P4`, `absent` → `ABSENT`

자동 함수 `migration_v11_attendance()` 제공.

### 5.2 TEACHERS role 갱신

자동 마이그레이션 함수 또는 운영자가 수동 수정:
- 김지헌, 최용수, 임성희, 곽조경, 황규일 → role = `admin`
- 윤희윤 → role = `newcomer_staff`
- 나머지 → role = `teacher`

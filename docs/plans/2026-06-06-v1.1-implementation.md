# v1.1 Implementation Plan — 권한·새가족·예배 출석·디자인

**Goal:** v1.0 위에 (1) 출석 enum 변경 (2) 권한 모델 (3) 새가족부 (4) 관리자 화면 (5) V0.3 디자인 적용을 한 sprint로 추가.

**Architecture:** v1.0 단일 HTML + GAS 그대로. 변경 = (a) GAS 액션 5개 추가 + 기존 액션 권한 게이트 강화 (b) 새 시트 1개 (NEWCOMER_PROGRESS) (c) ATTENDANCE 컬럼 1개 추가 (d) TEACHERS 컬럼 1개 추가 (title) (e) web/index.html 에 새 화면 3개 추가 + 디자인 보강.

**Spec:** [`../specs/2026-06-06-v1.1-roles-newcomers-services.md`](../specs/2026-06-06-v1.1-roles-newcomers-services.md)

## Phase A: 데이터 모델 변경 (시트 + GAS)

### A1. 시트 변경 (사용자 작업, 자동 안내)

- [ ] ATTENDANCE 컬럼 추가: F열에 "기타내용" 삽입 (기존 F, G 우측 1칸 이동)
- [ ] TEACHERS 컬럼 추가: E열에 "title"
- [ ] NEWCOMER_PROGRESS 탭 생성, 헤더 13개
- [ ] TEACHERS 6명 role/title 갱신

### A2. GAS Sheet.gs 확장

- [ ] SHEET_NAMES 에 `NEWCOMER_PROGRESS`, `RAW_CLASS` (= `2026 반편성`) 추가
- [ ] `getNextStudentId()` 함수: `2026 반편성` A컬럼 max + 1
- [ ] `appendToRawClass(rowObj)` 함수: 새가족 신규 행 추가
- [ ] `updateClassInRawSheet(studentId, newTeacher)` 함수: 등반 시 P컬럼 갱신 + 비고 태그 제거
- [ ] `lookupTeacher` 확장: title 컬럼도 응답에 포함
- [ ] `listStudentsForTeacher` — newcomer_staff/admin 권한 시 모든 학생 반환 옵션 추가
- [ ] `safeCachePut_` 그대로 사용 (이미 100KB 안전)

### A3. GAS Auth.gs 확장

- [ ] `authenticate(body)` 응답에 `permissions` 배열 포함:
  - admin → `['read_all', 'write_newcomer', 'graduate']`
  - newcomer_staff → `['write_newcomer', 'graduate']`
  - teacher → `[]`
- [ ] `hasPermission(auth, perm)` 헬퍼

### A4. GAS Actions.gs 확장

신규 액션:
- [ ] `getNewcomers` — newcomer_staff/admin: 새가족 명단 + NEWCOMER_PROGRESS 조인
- [ ] `addNewcomer` — newcomer_staff/admin: `2026 반편성`에 append (P='새가족', 비고 prefix '[새가족]')
- [ ] `setNewcomerProgress` — newcomer_staff/admin: NEWCOMER_PROGRESS UPSERT (week, completed, date)
- [ ] `graduate` — newcomer_staff/admin: `2026 반편성` P컬럼 갱신 + 비고 태그 제거 + NEWCOMER_PROGRESS 등반일 기록
- [ ] `getAllAttendance` — admin: 전체 반 출석 (대용량 주의)
- [ ] `getAllPrayers` — admin: 전체 반 기도제목
- [ ] `getAllRosters` — admin: 전체 반 명단 (byClass 그룹화)

기존 액션 수정:
- [ ] `setAttendance` — 새 enum 검증 + etcText 옵션
- [ ] `getAttendance` — admin은 classFilter='all' 가능
- [ ] `setAttendanceBatch` — 동일하게 enum + etcText

### A5. 새 마이그레이션 함수

- [ ] `migration_v11_attendance()` — 기존 상태 코드 s2/s3/s4/absent → P2/P3/P4/ABSENT 일괄 변환 + 컬럼 추가

  - newcomer_staff: 위 + **새가족부**
  - teacher: 출석 / 기도제목 / 기록 / 통계
- [ ] AppBar에 "관리자 · 강도사" 형태 부제 표시 (role + title 매핑)

### B3. 새가족부 화면 (NewcomersView)

- [ ] getNewcomers 호출 → 학생 카드 list
- [ ] 각 카드: 출석 5pill + 4주 체크박스 4개 + 등반 버튼 + 상세

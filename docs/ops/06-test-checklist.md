# E2E 테스트 체크리스트 (Spec Section 11)

배포 직후 다음 8가지를 차례로 확인. 한 항목이라도 실패하면 plan 의 해당 Task 로 돌아가 수정.

- [ ] **T1** TEACHERS 에 등록 안 된 이메일로 로그인 → "접근 권한이 없어요" 화면 표시
- [ ] **T2** 교사 A 로그인 → 본인 반 학생만 명단에 표시 (다른 반 학생 안 보임)
- [ ] **T3** 브라우저 콘솔에서 `fetch(GAS_URL, {method:'POST', headers:{'Content-Type':'text/plain'}, body: JSON.stringify({action:'setAttendance', idToken: sessionStorage.idToken, date:'2026-06-01', studentId:'<다른반 학생ID>', status:'s2'})}).then(r=>r.json()).then(console.log)` → `{ok:false, code:'forbidden'}`
- [ ] **T4** 같은 (날짜, 학생) 에 출석 → 결석 → 출석 토글. ATTENDANCE 시트에서 행 1개만 보이고 상태가 마지막 값
- [ ] **T5** 네트워크 끊고 출석 토글 → "네트워크 오류" 토스트 + 화면 상태 롤백
- [ ] **T6** 1시간 후 다시 로그인 시도 → silent refresh 또는 재로그인 모달, 작업 이어서 가능
- [ ] **T7** 시트에서 STUDENTS 에 새 학생 추가 → 앱 새로고침 시 5분 이내 반영 (캐시 만료)
- [ ] **T8** 시트에서 학생 `active=FALSE` → 명단에서 사라짐, 과거 ATTENDANCE 행은 유지

## v1.2 추가 시나리오 (2026-06-13)

- [ ] 로그인 직후 주 스테퍼가 이번 주일(예: "6월 2주 · 6/14 주일")을 기본값으로 표시한다
- [ ] ◀로 지난 주일 이동 시 헤더가 경고색(주황)으로 바뀌고 [이번 주로] 버튼이 보인다
- [ ] ▶로 미래 주일로는 이동할 수 없다
- [ ] 출석 탭 요약 바: 상태별 인원 + 미체크 N 표시, 탭하면 미체크 필터 토글
- [ ] 미체크 0이 되면 "출석 체크 완료! 🎉" 배너가 보인다
- [ ] 하단 일괄 버튼: "미체크 N명 → ○부 출석" 동작 + 실행취소 가능
- [ ] ▾에서 기타/결석 일괄 선택 시 사유 시트가 뜬다
- [ ] 현황 탭(주별 명단): 출석자 이름이 부별로, 기타는 사유와 함께 표시
- [ ] 현황 탭: 결석자 이름 옆 📞/💬 칩으로 전화/문자 앱이 열린다
- [ ] 현황 탭: 통계 세그먼트가 기존 통계와 동일하게 보인다
- [ ] 탭이 출석/현황/기도제목(+권한 탭)으로 표시된다
- [ ] 비행기 모드에서 칩 탭 → ⟳ 재시도 칩이 보이고, 네트워크 복구 후 탭하면 저장된다
- [ ] 학생 이름 탭 → 상세 시트에 기도제목·올해 출석 이력(최근 12회)이 보인다
- [ ] 다크 모드 토글 시 화면이 깨지지 않는다
- [ ] 안드로이드 크롬 + 아이폰 사파리에서 위 항목 재확인

## v1.3 추가 시나리오 (2026-06-13)

- [ ] 관리자 탭이 통계/새가족·등반/명단·기도 세그먼트로 나뉜다 (기본: 통계)
- [ ] 통계: 전체 평균 출석률 카드 + 최근 예배 인원이 표시된다
- [ ] 통계: 주별 추이 막대가 최근 12주까지 보이고 막대 위 인원 수가 맞다
- [ ] 통계: 반별 통계가 반 이름 가나다순으로, 학년별 통계가 중1→고3→기타 순으로 보인다
- [ ] 새가족·등반: 새가족 학생별 출석률이 보인다
- [ ] 새가족·등반: 등반 교육 1~4주 체크 상태가 새가족 탭과 일치한다 (읽기 전용)
- [ ] 새가족·등반: 올해 등반 완료자 이름·등반일·등반반이 보인다
- [ ] 새가족 탭에서 등반 처리 → 관리자 탭 등반 완료 목록에 즉시(캐시 무효화) 반영된다
- [ ] 명단·기도: 기존 반 선택 + 명단 + 기도제목이 그대로 동작한다
- [ ] 일반 교사 계정에는 관리자 탭이 보이지 않는다 (회귀 확인)
- [ ] 다크 모드에서 통계 카드·추이 차트가 깨지지 않는다
